## Research

**Accelerating Data Generation for Nonlinear Temporal PDEs via Homologous Perturbation in Solution Space**  
This project studies efficient construction of large-scale solution datasets for nonlinear time-dependent PDEs. The core idea is to perturb known PDE solution functions in the solution space and generate homologous variants at lower computational cost. Manuscript in preparation for AAAI 2027.

**Pretraining and Fine-tuning for Neural Operators**  
This project explores large-scale pretraining and parameter-efficient LoRA-style adaptation for neural operators. I worked on fine-tuning strategy design and preliminary validation on DPOT. Manuscript in preparation for NeurIPS 2026.

## Projects

**Personalized Mobile Agent**  
Built an LLM-based mobile agent using CoT, ReAct, ADB screenshots, and XML parsing to infer implicit user intent and execute mobile-device actions.

**Emotion-driven Virtual Human Assistant**  
Developed a virtual assistant pipeline that maps user text into emotion-aware video generation controls, combines Wan2.1-style video generation logic with talking-head synthesis, and received an A grade in the undergraduate innovation project.

**Personal Shopping AI Assistant Based on LLMs**  
Designed an external product database and system prompts for fuzzy intent recommendation and structured extraction. The project received awards in national LLM application competitions including AIGC Innovation Competition and Xinghuo Cup.
